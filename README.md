  - Walk ~6 min: [Topkapı Palace -> Gülhane Park](https://www.google.com/maps/dir/?api=1&origin=Topkapi+Palace,Istanbul&destination=Gulhane+Park,Istanbul&travelmode=walking)

## Getting around cheat sheet
- **IstanbulKart** for metro/tram/ferry/funicular.
- **Uber or BiTaksi** for cabs (match plate; check fare lands in the quoted range).
- **Hilton Maslak <-> city:** M2 "İTÜ-Ayazağa" -> Taksim (İstiklal) / Şişhane (Galata) / Vezneciler (old city), ~30-40 min.
- **Airport (IST):** M2 -> Gayrettepe -> M11, ~45 min.
- **Cash:** exchange small amounts at street stalls in the city, never the airport.
- **Restaurants:** skip places without posted prices; "tasting" plates aren't free; cash = no service fee/tax.
